## Where to search for more MBA-relevant projects yourself

**General project/dataset hubs**

* **Kaggle** (kaggle.com) — datasets + notebooks; filter by "Business" or "Retail" tags; also has a competitions section with real corporate-sponsored problems
* **GitHub Topics** (github.com/topics/...) — search topics like `supply-chain-analytics`, `rfm-analysis`, `six-sigma`, `market-basket-analysis`; single best way to browse by theme
* **UCI Machine Learning Repository** (archive.ics.uci.edu/ml) — source of the "Online Retail" dataset used in half the RFM projects above; solid, academically citable
* **Google Dataset Search** (datasetsearch.research.google.com) — searches across every public dataset host at once

**Business/India-specific data**

* **data.gov.in** — Indian government open data (industrial production, trade, MSME stats — useful to ground a project in Indian macro data)
* **IBEF** (ibef.org) — India Brand Equity Foundation; sector reports (automotive, FMCG, manufacturing), good for market-sizing sections
* **Statista** (statista.com) — India-specific market data (limited free tier, useful for citing numbers)

**Project walkthroughs / structured learning**

* **ProjectPro** (projectpro.io) — end-to-end business analytics project templates with data + code, some India-focused
* **Analytics Vidhya** (analyticsvidhya.com) — strong Indian analytics community, frequent RFM/forecasting/Six Sigma walkthroughs with downloadable code
* **Towards Data Science (Medium)** — search "case study" + your domain; many authors link their GitHub repo directly in the article

**Academic/citable sources (useful for your literature review section)**

* **Google Scholar** (scholar.google.com) — search "RFM segmentation retail" or "Six Sigma DMAIC manufacturing case study" for citable academic papers
* **SSRN** (ssrn.com) — management/operations working papers, good for MBA-level literature review citations

**Practical tip:** search GitHub directly using `site:github.com \\\\\\\\\\\\\\\[brand name] "power bi" OR "sql" OR "python" case study` — this is how I found the Zomato/Swiggy/Amazon/Walmart projects above, and it works for any other brand you want to target (Flipkart, Tata, Titan, Nestlé India, etc.).
